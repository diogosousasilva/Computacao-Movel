# 01 Overview

## Project Title
**Doggo: AI-Assisted Dog Image Explorer**

## Elevator Pitch
Doggo is a modern, Material Design 3 compliant Android application that connects users with an endless stream of dog pictures. Sourced from the public *Dog CEO API*, Doggo aims to provide a delightful and responsive interface for browsing, favoriting, and managing a personal collection of dog images locally.

## Problem Statement
In existing dog image apps, users often experience poor UI/UX, lack of offline functionality, and inability to save their favorite pictures for later viewing. Doggo solves this by persisting favorite images in a local SQLite database (Room) and delivering an elegant user experience.

## Why use AI?
Planning and architecting an application requires a robust set of files, classes, and UI structures. Using AntiGravity's AI allows us to accelerate the scaffolding process, establish a solid MVVM architecture right from the start, and generate boilerplate code quickly with fewer context-switching errors.
